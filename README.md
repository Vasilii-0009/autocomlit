Интсрукция по запуску проекта локально

**Клонируем репозиторий c помощью команды**

```
 $ git clone
```

**Устанавливаем зависимости**

```
$ npm i
```

**Запуск сервера в режиме разработки**

```
$ npm run dev
```

**Создание билда**

```
$ npm build
```

**Деплой проекта**

```
$ npm run deploy
```
