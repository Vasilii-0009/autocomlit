Инcтрукция по запуску проекта локально

**Клонируем репозиторий c помощью команды**

```
 $ git clone https://github.com/Vasilii-0009/autocomlit.git
```

**Устанавливаем зависимости**

```
$ npm i
```

**Запуск сервера в режиме разработки**

```
$ npm run dev
```

**Создание билда**

```
$ npm build
```

**Деплой проекта**

```
$ npm run deploy
```
